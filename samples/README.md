# Sample Documents (Fake)

Drop these into the matching demo tab to see each feature end-to-end.
**Everything here is fabricated** — names, dates, addresses, SSNs, and phone
numbers are not real. Phone numbers use the `555-01xx` reserved-for-fiction
block. SSNs use the impossible `000-xx-xxxx` prefix.

## `chronology/` — for the Chronology Builder

Three files from a fictional Texas custody matter, written out of date order
and contradicting each other on a couple of points (by design — the model
should flag inconsistencies). Upload all three at once.

Suggested "names to scrub" entry:
`Jane Marie Doe, John Q. Parker, Alex Parker, Morgan Parker`

## `correspondence/` — for the Correspondence Drafter

- `legalese_clause.txt` — paste into **"Translate legalese to plain English"**.
- `email_bullets.txt` — paste (or transcribe) into the bullets field of
  **"Draft an email"**.

## `templates/` — for the Template Generator

- `will_template.txt` — upload as the boilerplate.
- `client_facts.txt` — paste into the client-specific facts field.
